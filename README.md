



# 基于Springboot的网上摄影工作室

## Springboot-0098



## 技术栈

Springboot mybatisplus vue mysql maven



## 数据库表(10张)



## 功能介绍

```properties
网上摄影工作室管理员功能有个人中心，用户信息管理，作品分类管理，摄影作品管理，摄影圈，系统管理等。

用户可以注册登录，对摄影作品进行收藏和点赞以及评论，摄影圈发布信息，发布摄影作品，我的收藏等。
```
